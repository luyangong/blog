fzf简介

fzf是一个命令行模糊查询的工具。主要使用go语言实现。它提供一个友好的命令行模糊查询的界面，按照它的格式要求传参后，可以很好实现先搜索再操作，即"search and action"模式。这样就可以大大降低记忆负担，只需输入关键词即可完成各种操作。

安装fzf

# mac
brew install fzf

# ubuntu
apt install fzf
